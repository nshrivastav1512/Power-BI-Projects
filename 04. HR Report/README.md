## 📌 Project Overview

A comprehensive HR Analytics dashboard built in Power BI to visualize and analyze human resource data. This project provides insights into employee demographics, attrition patterns, and workforce metrics.

### 🚀 Key Features

- **Employee Demographics:** Visualize workforce composition by age, gender, and department
- **Attrition Analysis:** Identify patterns and factors contributing to employee turnover
- **Performance Metrics:** Track key HR performance indicators
- **Workforce Planning:** Data-driven insights for HR decision making

2. The data is embedded within the PBIX file
3. Explore the various dashboard pages and visualizations
