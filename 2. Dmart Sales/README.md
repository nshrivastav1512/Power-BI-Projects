## 📌 Project Overview

An interactive Power BI dashboard analyzing DMart's sales performance. This project provides insights into sales trends, product performance, and revenue metrics for one of India's leading retail chains.

### 🚀 Key Features

- **Sales Performance:** Monitor daily, monthly, and yearly sales trends
- **Product Analysis:** Track top-performing products and categories
- **Revenue Metrics:** Analyze revenue distribution and growth patterns
- **KPI Tracking:** Key performance indicators for business monitoring

---

## ⚙️ How to Run This Project

1. Open `Report/Dmart Dashboard.pbix` in Power BI Desktop
2. If prompted, update the data source path to point to `Dataset/DMart Sales Dashboard.xlsx`
3. Click **Refresh** to load the latest data
