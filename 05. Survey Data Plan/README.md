## 📌 Project Overview

A Power BI dashboard analyzing survey data from data professionals. This project visualizes insights about career paths, salaries, job satisfaction, and industry trends in the data field.

### 🚀 Key Features

- **Salary Analysis:** Compare salaries across different data roles and experience levels
- **Job Satisfaction:** Visualize satisfaction levels with work-life balance and salary
- **Career Insights:** Understand career transition paths into data
- **Industry Trends:** Analyze popular programming languages and tools

## ⚙️ How to Run This Project

1. Review the project documentation in `Dataset/Data Professional Survey Dashboard.docx`
2. Open the Power BI report in Power BI Desktop
3. If prompted, update the data source path to point to `Dataset/Survey Data.xlsx`
4. Click **Refresh** to load the latest data
