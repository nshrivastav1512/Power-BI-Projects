## 📌 Project Overview

A specialized dashboard for the DWR 3D Digitization project analyzing client data and order management. This project provides insights into order processing, client analytics, and project tracking.

### 🚀 Key Features

- **Order Analytics:** Track and analyze client orders
- **Project Tracking:** Monitor digitization project progress
- **Client Insights:** Analyze client data and trends
- **Custom Theming:** Uses custom Power BI theme for branding

## ⚙️ How to Run This Project

1. Open `3D Digitization project.pbix` in Power BI Desktop
2. If prompted, update the data source path to point to the Excel files
3. Click **Refresh** to load the latest data
4. Optionally, apply the custom theme from `3d theme.json`
