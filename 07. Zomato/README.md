## 📌 Project Overview

An interactive Power BI dashboard analyzing Zomato restaurant data. This project provides insights into restaurant ratings, cuisines, pricing, and geographic distribution of restaurants on the Zomato platform.

### 🚀 Key Features

- **Restaurant Ratings:** Analyze rating distributions and top-rated restaurants
- **Cuisine Analysis:** Explore popular cuisines and their performance
- **Price Analysis:** Compare pricing across different restaurant types
- **Geographic Insights:** Visualize restaurant distribution across locations

## ⚙️ How to Run This Project

1. Open `Reports/Zomato restaurants analysis dashboard in Power BI.pbix` in Power BI Desktop
2. If prompted, update the data source path to point to `Dataset/zomato_data.xlsx`
3. Click **Refresh** to load the latest data
