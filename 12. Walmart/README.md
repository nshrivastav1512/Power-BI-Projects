## 📌 Project Overview

A retail analytics dashboard analyzing Walmart sales data. This project provides insights into sales performance, store comparisons, and product category analysis for one of the world's largest retailers.

### 🚀 Key Features

- **Sales Performance:** Analyze sales trends across stores and regions
- **Store Comparison:** Compare performance metrics between stores
- **Category Analysis:** Track product category performance
- **Seasonal Trends:** Identify sales patterns based on seasonality

## ⚙️ How to Run This Project

1. Open the Power BI report in Power BI Desktop
2. Update the data source path to point to the Dataset folder
3. Click **Refresh** to load the data
