## 📌 Project Overview

A comprehensive supply chain analytics dashboard for the automotive industry. This project analyzes car supply chain data to optimize inventory management, track delivery metrics, and identify bottlenecks in the supply chain.

### 🚀 Key Features

- **Inventory Analysis:** Monitor stock levels and inventory turnover
- **Delivery Metrics:** Track delivery times and performance
- **Supplier Analysis:** Evaluate supplier performance and reliability
- **Cost Optimization:** Identify areas for cost reduction in the supply chain

## ⚙️ How to Run This Project

1. Review the business requirements in `Dataset/BRD-FRD Document.docx`
2. Open `Reports/PBI File.pbix` in Power BI Desktop
3. If prompted, update the data source path to point to `Dataset/Car_SupplyChainManagementDataSet.csv`
4. Click **Refresh** to load the latest data
