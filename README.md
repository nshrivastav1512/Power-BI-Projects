## 🛠️ Technologies & Skills

### **Tools Used**
*   Microsoft Power BI Desktop
*   SQL Server / SSMS
*   Microsoft Excel & CSV Processing
*   Power Query (M) for ETL
*   DAX (Data Analysis Expressions)
*   Python (pandas, PyPDF2) for data extraction

### **Skills Demonstrated**
*   ✅ **Data Modeling:** Star Schemas, Snowflake Schemas, and complex relationship building.
*   ✅ **Data Transformation:** Robust ETL pipelines via Power Query and SQL.
*   ✅ **Advanced DAX:** Complex row-context iterations, dynamic thresholds, and time-intelligence.
*   ✅ **UI/UX Design:** Immersive transparent overlays, animated HTML/CSS backgrounds, and custom themes.
*   ✅ **AI-Powered Insights:** Key Influencers, Decomposition Trees, and forecasting.

## 🚀 Getting Started

### Prerequisites
1.  [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (Free download)
2.  SQL Server (Required for `08. Retail` generating mock databases)
3.  Microsoft Excel (For viewing basic source data files)

### Running a Project
1.  Navigate to the specific project folder you wish to explore.
2.  Open the `.pbix` file in Power BI Desktop.
3.  Update data source connections via **Transform Data -> Data Source Settings** if prompted (especially for SQL-based projects).
4.  Click **Refresh** to load the data model and view the dashboard!
