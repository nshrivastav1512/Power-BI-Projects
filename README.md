## 🚀 Getting Started

### Prerequisites
- [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (Free download)
- SQL Server (for projects with SQL backend)
- Microsoft Excel (for viewing source data)

### Running a Project
1. Navigate to the project folder
2. Open the `.pbix` file in Power BI Desktop
3. Update data source connections if prompted
4. Click **Refresh** to load data
5. Explore the interactive dashboard!
