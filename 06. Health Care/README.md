## 📌 Project Overview

A comprehensive healthcare analytics dashboard that visualizes patient data, hospital operations, and healthcare metrics. This project provides actionable insights for healthcare administrators and decision-makers.

### 🚀 Key Features

- **Patient Analytics:** Analyze patient demographics and visit patterns
- **Operational Metrics:** Monitor hospital capacity and resource utilization
- **Treatment Analysis:** Track treatment outcomes and patient flow
- **Financial Insights:** Analyze healthcare costs and billing metrics
