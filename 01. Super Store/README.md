## 📌 Project Overview

A comprehensive Power BI dashboard analyzing Super Store sales data to provide actionable business insights. This project visualizes sales performance, profitability, and customer trends across multiple dimensions.

### 🚀 Key Features

- **Sales Analysis:** Track revenue, quantity sold, and order metrics
- **Profitability Insights:** Analyze profit margins across categories and regions
- **Customer Segmentation:** Understand buying patterns by customer segment
- **Geographic Analysis:** Visualize sales distribution across regions

---

## ⚙️ How to Run This Project

1. Open `Report/Super Store Dashboard.pbix` in Power BI Desktop
2. If prompted, update the data source path to point to `Data/superstore.xlsx`
3. Click **Refresh** to load the latest data
