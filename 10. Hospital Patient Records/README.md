## 📌 Project Overview

A healthcare operational dashboard analyzing hospital patient records. This project provides insights into patient admissions, treatment patterns, hospital capacity utilization, and healthcare outcomes.

### 🚀 Key Features

- **Patient Demographics:** Analyze patient age groups, gender distribution, and admission types
- **Admission Analytics:** Track admission/discharge patterns and length of stay
- **Department Analysis:** Monitor workload across different hospital departments
- **Operational Insights:** Healthcare performance metrics and KPIs

## ⚙️ How to Run This Project

1. Open `Reports/Health Care Data.pbix` in Power BI Desktop
2. If prompted, update the data source path to point to the Dataset folder
3. Click **Refresh** to load the latest data
4. Review the project report for detailed documentation
