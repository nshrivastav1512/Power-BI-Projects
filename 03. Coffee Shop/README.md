## 📌 Project Overview

A dynamic Power BI dashboard for coffee shop sales analysis. This project provides comprehensive insights into coffee shop operations, including sales patterns, popular products, and peak business hours.

### 🚀 Key Features

- **Sales Trends:** Analyze daily and hourly sales patterns
- **Product Performance:** Identify best-selling coffee and food items
- **Revenue Analysis:** Track revenue by product category and time period
- **Customer Insights:** Understand customer purchasing behavior

## ⚙️ How to Run This Project

1. Open `Report/CoffeeShopDashboard.pbix` in Power BI Desktop
2. If prompted, update the data source path to point to `DataSet/Coffee Shop Sales.xlsx`
3. Click **Refresh** to load the latest data
